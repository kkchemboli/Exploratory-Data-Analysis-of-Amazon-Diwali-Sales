# Exploratory Data Analysis of Amazon Diwali Sales

## Kaggle Dataset
[Amazon Product Sales 2025](https://www.kaggle.com/datasets/anandshaw2001/amazon-product-sales-2025)

## In This Exploratory Analysis
- Explore and visualize the data using **Matplotlib** and **Seaborn**  
- Clean and preprocess the dataset  
- Explore features and understand distributions  
- Find correlations between features and visualize them  
- Answer key questions about the data, such as:
  1. Which two Product Category has had the highest total unit sales?
  2. Which Category has the highest undelivered orders?
  3. Which Product Category is the most famous in each state?
  4. Were Books sold more in Karnataka or Maharashtra?
  5. Which Method of payment was used the most in Sikkim?

## Tools & Libraries
- **Python** 
- **Pandas** – data manipulation  
- **NumPy** – numerical operations  
- **Matplotlib** – visualization  
- **Seaborn** – advanced visualization  

## Steps in the Analysis
1. **Data Loading & Inspection** – Check missing values, data types, and unique values  
2. **Data Cleaning** – Handle missing/inconsistent data and convert column types  
3. **Exploratory Analysis** – Summarize data with groupby, aggregation, and descriptive stats  
4. **Visualization** – Bar charts, histograms, and heatmaps for trends and correlations  
5. **Insights** – Identify top-selling products, states, and correlations between features  

## Conclusion
This analysis provides insights into **Amazon's Diwali sales in 2025**, highlighting top-performing products, high-sales regions, and overall trends that can inform **inventory planning, marketing, and business strategy**.
